Fajlovi u ovom folderu se ponašaju kao serveri.
Kod testiranja pub-weather, i pub-pollution se proverava cela poruka, 
dok se kod ostalih proverava samo format poruke.

Uputstvo za pokretanje:
Na primer za testiranje pub-weather:
    python3 pub-weather-test.py
Zatim se treba pozicionirati u src folder i pokrenuti:
    python3 pub-weather-test.py

Opšti slučaj:
    python3 pub-name-test.py
Nakon pozicioniranja u src folder:
    python3 pub-name.py
